Command Golang

Setiap pengembangan aplikasi dengan bahasa pemrograman apapun selalu punya command line untuk menjalankan program yang ditulis menggunakan bahasa pemrograman tertentu. Salah satunya pada Golang beberapa command line yang sering dipakai dalam development adalah sebagai berikut :

Go run

Digunakan untuk mengeksekusi file go atau untuk menjalankan program yang dibuat dengan Go. penulisan command ini bisa di lakukan di terminal dengan mengarah pada direktori file go tersebut berada

Output dari perintah diatas adalah hasil dari program go yang ditulis di file main.go tersebut

Go build

Ketika eksekusi program menggunakan Go run terjadi proses kompilasi. File hasil kompilasi akan disimpan pada folder temporary untuk selanjutnya langsung dieksekusi.

Berbeda dengan go build , command ini menghasilkan file executable atau binary pada folder yang sedang aktif

Executable atau binary dari hasil kompilasi tersebut mengikuti format executable pada OS nya masing-masing. Jika menggunakan OS Windows maka format binary yang muncul adalah main.exe, begitu juga dengan linux atau mac OS akan mengikuti executable pada OS nya.

Go test​

Go menyediakan package testing yang berguna untuk keperluan unit test. File yang digunakan untuk testing harus memiliki akhiran _test.go agar Go dapat mengenali bahwa file tersebut dipakai sebagai unit test.

Go mod

Untuk memulai membuat project go, anda perlu membuat module nya terlebih dahulu dengan menggunakan go mod init [nama package project]. Command ini yang akan menampung module atau package yang dibutuhkan dalam pembuatan project

Ketika command go mod di buat, maka file go.mod akan terbentuk. file didalam require adalah beberapa module atau package yang di download dengan menggunakan command go get.

Go get

Go dapat mendownload beberapa package yang disediakan dari sumber luar. Untuk dapat mengambil package dari luar tersebut anda dapat menggunakan command go get [url package].

Command go get tidak bisa dilakukan ketika anda belum membuat package modulnya atau belum melakukan perintah go mod. Karena pada dasarnya tempat penyimpanan module yang anda download itu ada pada go.mod.

Go help

Masih banyak command go yang bisa anda temukan sendiri dan dapat anda cobain sendiri dengan mengetikan command line go help, dan semua command yang dapat dilakukan oleh muncul beserta penjelasannya.
