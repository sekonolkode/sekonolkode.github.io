Menginstall Golang

Sebelum belajar pemrograman Golang, kita perlu menginstall installer Go nya terlebih dahulu. Untuk dapat mendownload installer Go bisa melalui website resminya yaitu go.dev.

Instalasi Go di Windows

Setelah mendownload installer Go, jalankan installer tersebut, klik next hingga proses instalasi selesai. Secara default jika anda tidak merubah path pada saat instalasi, Go akan terinstall di C:\go. Path tersebut secara otomatis akan didaftarkan dalam PATH environment variable di laptop anda.

Untuk mengecek apakah GO sudah berhasil di install, silahkan buka Command Prompt / CMD, eksekusi perintah go version

  • Jika output adalah sama dengan versi Go yang ter-install, menandakan proses instalasi berhasil.
  • Sering terjadi, command go version tidak bisa dijalankan meskipun instalasi sukses. Solusinya bisa dengan restart CMD (tutup CMD, kemudian buka lagi). Setelah itu coba jalankan ulang command go version
  • Jika masih belum muncul version dari Go nya, coba cek pathnya kembali di environment variable pastikan sudah di setup di pathnya

Instalasi Go di MacOS

Cara termudah instalasi Go di MacOS adalah menggunakan Homebrew. Install terlebih dahulu Homebrew (jika belum ada) cara install homebrew ada pada websitenya yaitu Homebrew

Jika sudah terinstall homebrew nya anda bisa langsung eksekusi perintah berikut di terminal

  • brew install go
  • Tambahkan path binary Go ke PATH environment variable : echo "export PATH=$PATH:/usr/local/go/bin" >> ~/.bash_profile kemudian ketikan perintah : source ~/.bash_profile
  • Jalankan perintah berikut mengecek versi Go : go version
  • Jika outputnya adalah sama dengan versi Go yang ter-install, menandakan proses instalasi berhasil.
  • Jika outputnya bukan versi Go, coba bisa lakukan kembali proses instalasinya dari awal

Instalasi Go di Linux​

Unduh arsip installer dari go.dev, pilih installer untuk Linux yang sesuai dengan jenis bit komputer anda. Atau anda juga bisa download lewat CLI dengan menggunakan wget atau curl seperti wget https://storage.googleapis.com /golang/go1.19.src.tar.gz

  • Buka terminal, extract arsip tersebut ke /usr/local
  • tar -C /usr/local -xzf go1.19.src.tar.gz
  • Tambahkan path binary Go ke PATH environment variable.
  • echo "export PATH=$PATH:/usr/local/go/bin" >> ~/.bashrc
  • source ~/.bashrc

Selanjutnya, eksekusi perintah berikut untuk mengetes apakah Go sudah terinstal dengan benar go version.

  • Jika output adalah sama dengan versi Go yang ter-install, menandakan proses instalasi berhasil.
  • Jika outputnya bukan versi Go, coba bisa lakukan kembali proses instalasinya dari awal
